Raspberry Pi Zero のピン配列

GPIO, 電源, GND, I2C 信号線などのピン配列を記載します。

  • 白い文字で書かれたピンだけが使えます
  • GND、3.3V、5V はそれぞれ電源とグランドです
  • 数字 + PD||PU と書かれているピンは GPIO 端子
    • PD:プルダウン, PU:プルアップ
  • SCL, SDA は I2C インターフェースのピンです
